# Hello World - macOS 터미널 실행 가이드

## 개요
macOS 터미널에서 실행 가능한 Python Hello World 스크립트입니다.

## 실행 방법

### 1. 터미널 열기
- Spotlight(Command + Space) 에서 "터미널" 검색 후 실행
- 또는 응용 프로그램 > 유틸리티 > 터미널

### 2. 프로젝트 디렉토리로 이동
```bash
cd /workspace/proj-20260418-helloworld-guide
```

### 3. 스크립트 실행
```bash
python3 hello.py
```

### 4. 예상 출력
```
Hello World
```

## 요구사항
- macOS 환경
- Python 3.x 설치됨 (기본 포함)

## 확인 사항
Python 설치 여부 확인:
```bash
python3 --version
```
